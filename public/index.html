Joyeria Ramos

API RESTful

Código fuente en https://github.com/Rafa567/joyeria-ramos

Se proporciona la siguiente API:

Catalogos

/api/catalogos    POST
Crear producto catálogo
/api/catalogos    GET ALL
Obtener todos los productos del catálogo.
/api/catalogos/id    GET
Obtener producto del catálogo
/api/catalogos/id    PUT
Modificar producto del catálogo
/api/catalogos/id    DELETE
Eliminar producto del catálogo

CLIENTES

/api/clientes    POST
Crear cliente
/api/clientes    GET ALL
Obtener todos los clientes.
/api/clientes/id    GET
Obtener cliente
/api/clientes/id    PUT
Modificar cliente
/api/clientes/id    DELETE
Eliminar cliente
